Tela de Cadastro
A tela de cadastro é destinada a novos usuários que desejam se tornar clientes da confeitaria. Ao se cadastrar, o usuário obtém acesso à plataforma, incluindo funcionalidades como visualização do cardápio, realização de pedidos, acompanhamento de entregas e outros recursos.

Após o cadastro, o usuário poderá acessar a plataforma utilizando suas credenciais de login.

Campos do Cadastro
Nome Completo: Nome completo do usuário.

E-mail: Endereço de e-mail válido, utilizado para cadastro e login. O sistema não permite o uso de um e-mail já cadastrado.

Senha: Senha de acesso à plataforma, com no mínimo 6 caracteres.

Ações Disponíveis
Criar Conta: Registra o usuário na plataforma da confeitaria.

Entrar: Redireciona usuários já cadastrados para a tela de login, caso tenham acessado a tela de cadastro por engano.

Informações Adicionais
O sistema é voltado exclusivamente para confeitarias que trabalham com encomendas, portanto, não é necessário fornecer informações adicionais, como endereço ou outros dados pessoais, no momento do cadastro.

A plataforma está disponível para qualquer usuário, seja ele um cliente da confeitaria ou o próprio confeiteiro que adquiriu o software. Inicialmente, todos os novos cadastros são registrados como clientes. Caso seja necessária a mudança para administrador, o usuário deverá entrar em contato com o suporte para solicitar a alteração.
