Tela de Login
A tela de login é a porta de entrada para a plataforma da confeitaria. Para acessar o sistema, o usuário deve utilizar as credenciais cadastradas previamente, inserindo seu e-mail e senha.

Após a autenticação bem-sucedida, o usuário será redirecionado para a página inicial da plataforma.

Campos de Login
E-mail: Endereço de e-mail utilizado no momento do cadastro.

Senha: Senha definida durante o cadastro.

Ações Disponíveis
Entrar: Autentica o usuário e o direciona para as telas do sistema.

Cadastre-se: Redireciona novos usuários para a tela de cadastro.

Ver a senha: Ao clicar no ícone de olho, é possível mostrar ou esconder a senha que foi digitada.

Esqueceu a senha: Ao clicar em esqueceu a senha, o fluxo é o seguinte: primeiramente o usuário digita um e-mail. Depois, é verificado se esse e-mail está na base de dados da confeitaria. Caso esteja, é enviado um e-mail para esses destinatário, com um código de 6 dígitos. A própria tela, após enviar o e-mail, já abre a tela de verificação de código para informar o código recebido por e-mail. Mas caso o usuário queira, pode diretamente no e-mail clicar num link redirecionador, que já abre a tela de verificação de código com o código preenchido. Além disso, o código pode ser digitado manualmente ou colado. Com o código preenchido, é verificado na base de dados se existe algum usuário com o código informado. Além disso, é verificado se esse código não expirou (Já que possui um tempo útil de 15 minutos) e se ele já não foi usado, pois cada código pode ser utilizado apenas uma vez. Após todas essas validações forem validadas com sucesso, o usuário é redirecionado para uma página de redefinição de senha. Após a senha ser redefinida, ele é redirecionado para a página de login, onde pode concluir o login normalmente com a nova senha definida.
