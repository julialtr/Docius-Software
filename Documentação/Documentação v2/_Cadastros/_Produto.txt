Tela de Cadastro de Produtos

Essa tela permite cadastrar, editar e excluir produtos que farão parte do cardápio da confeitaria. Cada produto deve estar vinculado a uma receita, e seu preço é definido com base na precificação.

Funcionalidades
Lista de Produtos
- Exibe todos os produtos cadastrados, apresentando: Nome, Preço (calculado a partir da precificação), Receita vinculada
- Todas essas informações podem ser ordenadas de forma crescente ou decrescente.
- Busca por produto: É possível localizar um produto utilizando um campo de pesquisa.

Visualização de Receita Vinculada
- Ao clicar na seta '>', é exibido um resumo da receita associada ao produto, contendo: Nome da receita, Tempo de preparo, Rendimento (quantidade de porções), Lista de ingredientes
- Cada ingrediente é exibido com: Nome da categoria, Medida necessária, Unidade de medida

Cadastro de Novo Produto
- Para adicionar um novo produto, basta clicar no botão "+".
- Os campos obrigatórios são: Nome do produto, Receita vinculada
- É preciso que a receita seja cadastrada antes do produto, para no cadastro de produto, apenas fazer o vínculo.
- O preço do produto é calculado automaticamente posteriormente, com base na precificação.

Edição e Exclusão de Produtos
- Os produtos podem ser editados ou removidos conforme necessário.

Importância do Cadastro de Produtos
- O cadastro de produtos é essencial para a montagem do cardápio da confeitaria.
- Sem um produto cadastrado, não é possível disponibilizá-lo para os clientes na plataforma.
