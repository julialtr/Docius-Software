Tela de Cadastro de Receitas

Acesso restrito: Disponível apenas para usuários administradores.

Esta tela permite cadastrar, editar ou remover receitas para a confeitaria. As receitas estão diretamente vinculadas à precificação e ao cardápio, sendo essenciais para o funcionamento da plataforma.

Funcionalidades
Lista de Receitas
- Exibe todas as receitas cadastradas na confeitaria.
- Busca por receita: É possível buscar uma receita utilizando um campo de pesquisa.
- A lista contém as seguintes informações: Nome, Descrição, Tempo de preparo, Rendimento (quantidade de porções).
- Possibilidade de ordenar os dados de forma crescente ou decrescente.

Visualização de Ingredientes da Receita
- Ao clicar na seta '>' ao lado de uma receita, é exibida a lista de ingredientes da receita.
- Para cada ingrediente, são apresentadas as seguintes informações: Nome da categoria de ingrediente, Medida utilizada, Unidade de medida

Cadastro de Nova Receita
- Para adicionar uma nova receita, basta clicar no botão "+".
- Os campos obrigatórios para cadastro de uma receita são: Nome, Tempo de preparo, Quantidade de porções, Lista de ingredientes
- Descrição da receita é opcional.

Cadastro de Ingredientes na Receita
- Ao adicionar ingredientes, é necessário:
- Escolher a categoria de ingrediente (vinculada ao cadastro de categorias de ingredientes).
- Definir a medida e a unidade de medida.
- Unidades de medida pré-cadastradas: "g", "kg", "ml" e "l".
- Restrições: Não é permitido adicionar a mesma categoria de ingrediente mais de uma vez na mesma receita.

Edição e Exclusão de Receitas
- As receitas podem ser editadas ou removidas conforme necessário.

Restrições de Exclusão
- Como as receitas estão vinculadas a categorias de ingredientes, não é possível excluir uma categoria de ingrediente se ela estiver associada a alguma receita.
